# Descripción del Proyecto

MediCitas es una aplicación web orientada a la administración de citas médicas. 

El sistema permite que los pacientes puedan:

- Registrarse e iniciar sesión
- Buscar doctores por especialidad
- Agendar citas médicas
- Consultar historial médico
- Administrar sus citas

El proyecto fue desarrollado utilizando:

## Tecnologías Utilizadas

### Backend
- Node.js
- Express.js
- MySQL
- JWT
- bcryptjs

### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

# Módulos del Sistema

## 1. Usuarios

Módulo encargado de la administración de usuarios del sistema.

Funciones:
- Registrar usuarios
- Consultar usuarios
- Actualizar información
- Eliminar usuarios

---

## 2. Doctores

Módulo encargado de la gestión de médicos y especialistas.

Funciones:
- Registrar doctores
- Consultar doctores
- Buscar por especialidad
- Actualizar información
- Eliminar doctores

---

## 3. Especialidades

Módulo encargado de administrar las especialidades médicas.

Funciones:
- Registrar especialidades
- Consultar especialidades
- Actualizar especialidades
- Eliminar especialidades

---

## 4. Citas

Módulo principal del sistema para la gestión de citas médicas.

Funciones:
- Crear citas
- Consultar citas
- Actualizar citas
- Cancelar citas
- Consultar citas pendientes

---

## 5. Historial Médico

Módulo encargado del historial clínico de los pacientes.

Funciones:
- Registrar historial médico
- Consultar historial
- Actualizar historial
- Eliminar historial
- Buscar historial por paciente
